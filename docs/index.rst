COSMO
=====
COSMO (COS Monitoring) supports the efforts of the Cosmic Origins Spectrograph (COS) team at the 
Space Telescope Science Institute (STScI) in Baltimore, MD. COSMO uses a MySQL database server on the
backend to store processed COS data that is used to track trends that effect the instrument performace.
The objective of the this project is to automate data processing and trend tracking for the COS team
and community to have the most up-to-date resources available to extend the lifetime of COS.
